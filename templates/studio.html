How to send emails

  1. Generate certificates first using the Generate button.
  2. Use sender email and app password (for Gmail, create a 16-character App Password).
  3. Keep security as TLS unless your SMTP provider says otherwise.
  4. Click Send Emails to deliver one certificate per CSV row email.
