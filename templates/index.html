The Creator

Aaditya Devadiga

I built this because running multiple club events made one thing obvious: manual certificate creation keeps stealing time from the work that actually matters.

As president of my club, I kept seeing the same problem repeat. Every event needed certificates, every batch took effort, and existing systems asked for money while still limiting what could be customized. So I decided to build my own generator instead.

That gave me two things I cared about most: speed for repeated event work, and freedom to create any layout, workflow, or feature I want without waiting on someone else's platform.
