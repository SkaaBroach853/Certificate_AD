CertFlow Studio Help

Full guide from basic setup to sending emails.

1. Overview

This tool helps you create certificates in bulk. You upload a template, load recipient data, place text boxes, preview the result, generate downloads, and optionally send the certificates by email.

2. Setup Files

Template: Upload a PNG or JPG certificate background.

Recipient data: Upload a CSV file or paste CSV text and click Use Pasted Data.

Example CSV:

Name,Email,Course,Date
John Doe,john@example.com,Python,19 Apr 2026
Jane Smith,jane@example.com,Design,19 Apr 2026

3. Design Section

Click Add Text Box to place a new text layer on the certificate.

You can drag each box anywhere and resize it from the bottom-right corner.

Example lines:

Certificate of Completion
Awarded to {Name}
For successfully completing {Course}

Use placeholders from your CSV, such as {Name}, {Email}, {Course}, {Date}, and {CertificateID}.

4. Output Section

Choose PNG or PDF output. Set the filename template if you want custom names.

Example filename template:

{Name}_{CertificateID}

You can also enable QR codes and control their position and size.

  1. Preview: Generates one sample certificate.
  2. Generate: Creates all certificates as a ZIP.
  3. Generate One: Creates one certificate for the selected row number.

5. Email Section

Use this when you want the app to send certificates directly to recipients.

Usually required: SMTP host, SMTP port, SMTP security, sender email, sender password or app password, subject, and body.

Gmail example:

SMTP Host: smtp.gmail.com
SMTP Port: 587
SMTP Security: TLS
Sender Email: yourname@gmail.com
Password: Google App Password
  1. Email Preview: Shows how the email will look.
  2. Test SMTP: Checks whether login and SMTP settings work.
  3. Send Emails: Generates certificates and emails all rows from the CSV.

6. Recommended Workflow

  1. Upload the certificate template.
  2. Upload or paste the CSV data.
  3. Add and style text boxes.
  4. Preview a sample.
  5. Generate all certificates or send them by email.

7. Common Issues

No CSV data loaded: Make sure the pasted or uploaded CSV has a header row and at least one data row.

Preview failed: Usually means template, CSV, or text box data is missing.

Email failed: Usually means wrong SMTP settings, wrong password, or missing email values in the CSV.
